How this lesson works

We will build the clicker game in small steps. Each step explains one part of the page, then checks your code before moving on.

  • HTML creates the text and buttons you see on the page.
  • JavaScript makes the buttons do things when you click them.
  • The score is the number that grows when you click the button.
  • The upgrade button spends points to make each click stronger.
  • The reset button starts the game over.

How to code along

  • Click Load starter code to get the beginning page structure.
  • Read the step text at the top. It tells you exactly what to do next.
  • Type the code written in the comments or the hint text.
  • Click Run step to check if the step is finished.
  • If it says the step is not complete, just follow the message and try again.

What you can change

The editor contains a page made of three parts:

  • HTML in the <body> section – this makes the game page and buttons.
  • CSS in the <style> block – this changes how the game looks.
  • JavaScript in the <script> block – this makes the buttons do things.

For this lesson, follow the step text and use the starter comments as your guide.
